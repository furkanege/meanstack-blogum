MongoDB

MongoDB, ücretsiz ve açık kaynaklı bir çapraz platform belge yönelimli veritabanı programıdır. Bu NoSQL veritabanı programı şemaları olan JSON benzeri belgeleri kullanır.

ExpressJS

Express.js bir Node.js çerçevesidir. Node.js, web ve ağ uygulamaları oluşturmak için JavaScript'in Web Tarayıcılarının dışında kullanılmasına izin veren bir platformdur.

Angular

Angular, mobil ve masaüstü web uygulamaları oluşturmak için bir geliştirme platformudur.

NodeJS

Node.js, Chrome'un V8 JavaScript motoru üzerine kurulmuş bir JavaScript çalışma zamanıdır.
